# mycache
分布式LRU缓存，通过key-value存储。支持并发访问，通过Protobuf通信。  
+ 通过key选择该访问的节点
+ 缓存获取不到时调用用户接口进行缓存填充，通过LRU机制进行缓存淘汰
+ 每个key对应唯一节点，数据仅在一台节点上缓存，保证集群不会被热点数据占满
+ 不支持缓存更新，不支持定时淘汰
+ 不需要额外的客户端，客户端本身也是缓存服务器，与自身进行通信
+ 可以分布式部署，也可以单机多端口部署

## 实现
1. 实现LRU，加入锁以达到并发安全
3. 一致性哈希选择节点，避免缓存雪崩
4. 实现同时运行多个缓存实例，封装缓存填充接口
5. 实现单点反射，防止缓存被击穿
6. 使用protobuf通信
